# Screenshots Directory

This directory contains screenshots of the CleanME application showcasing its modern macOS design and features.

## Required Screenshots

Please add the following screenshots to this directory:

1. **main-interface.png** - Main application interface showing the scan view
2. **scan-results.png** - Results view with file listings and deletion options
3. **settings-panel.png** - Settings view with all configuration options
4. **about-credits.png** - About view showing credits and branding

## Screenshot Guidelines

- **Resolution**: Use 2x Retina resolution (e.g., 2000x1400px)
- **Format**: PNG with transparency where applicable
- **Quality**: High quality, no compression artifacts
- **Content**: Show realistic data, avoid empty states
- **Theme**: Capture both light and dark mode if possible

## File Naming Convention

- Use lowercase with hyphens
- Be descriptive but concise
- Include version number if multiple versions exist
