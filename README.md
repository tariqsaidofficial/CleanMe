# CleanME

CleanME is a macOS application designed to safely and transparently clean cache and log files to improve performance and manage storage space without compromising system integrity. 

## Features

- **Safe Cleanup**: Cleans temporary files (cache/logs) while ensuring system files remain untouched.
- **User-Friendly Interface**: Built with SwiftUI, providing a modern and intuitive user experience.
- **Detailed Reports**: Offers detailed reports on what will be deleted and how much space will be freed.
- **Settings Management**: Allows users to configure options related to the app's functionality.
- **Scheduled Cleanup**: Supports scheduling cleanups for regular maintenance.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/CleanME.git
   ```
2. Navigate to the project directory:
   ```
   cd CleanME
   ```
3. Open the project in Xcode:
   ```
   open CleanME.xcodeproj
   ```
4. Build and run the application.

## Usage

- Launch the application.
- Navigate to the **Scan** view to initiate a file scan.
- Review the results in the **Results** view.
- Adjust settings in the **Settings** view as needed.
- Confirm and proceed with the cleanup process.

## Privacy

CleanME is designed with user privacy in mind. It does not collect personal data by default and adheres to best practices for data protection.
